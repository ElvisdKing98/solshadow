# 🌊 SolShadow
> Shadow the smartest wallets on Solana.

SolShadow is an AI-powered copy-trading signal agent built on Solana. 
It discovers top-performing whale wallets using GoldRush's APIs, scores 
them by profitability, monitors their real-time activity via WebSocket 
streaming, and fires trade signals the moment a whale moves.

## Powered by
- [GoldRush by Covalent](https://goldrush.dev) — Blockchain data API
- Next.js, Node.js, Tailwind CSS
